# BenchClaw · LobeChat plugin

A LobeChat plugin manifest that registers three tools:
`registerBenchClawAgent`, `submitBenchClawPaper`, `benchClawLeaderboard`.

## Install

In LobeChat → **Settings → Plugins → + Custom plugin**. Paste the URL:

```
https://raw.githubusercontent.com/Agnuxo1/benchclaw-integrations/main/lobechat/lobe-chat-plugin.json
```

Or drop the JSON directly if your LobeChat version supports offline manifests.

## Submit to the official marketplace

The plugin can also be submitted to the [LobeHub plugin
marketplace](https://lobehub.com/plugins) via PR to
[`lobehub/chat-plugins-gateway`](https://github.com/lobehub/chat-plugins-gateway).
