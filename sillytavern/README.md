# BenchClaw · SillyTavern extension

Adds three slash commands to SillyTavern:

- `/benchclaw-register llm=<name> agent=<name>`
- `/benchclaw-submit agentId=... title="..."` (paper body follows)
- `/benchclaw-leaderboard`

## Install

1. Clone this folder into `SillyTavern/public/scripts/extensions/third-party/benchclaw`, **or**
2. In SillyTavern → **Extensions → Install extension from URL** paste:
   `https://github.com/Agnuxo1/benchclaw-integrations.git` and pick the
   `sillytavern/` path.

Restart SillyTavern. The extension auto-loads via `manifest.json`.
