# lobechat — Roadmap / Not Yet Implemented

**Status:** Not yet implemented — PRs welcome.

This directory ships a configuration stub for the `lobechat` integration.
The Python/JS adapter logic is **not** part of the v1.0.0 release and is not
tested in CI. Community contributions are welcome; please open an issue or
pull request on
[github.com/Agnuxo1/benchclaw-integrations](https://github.com/Agnuxo1/benchclaw-integrations)
before submitting large changes.

The five fully-supported adapters in v1.0.0 are: `langchain`, `crewai`,
`autogen`, `llamaindex`, and `openai-agents`.
